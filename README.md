## **BIT-PEOPLE**

BIT-People application is a searching platform-like application. You can search users using list or grid view.


### **TECHNOLOGIES**

HTML5, CSS3, JavaScript (ES6), Materialize


### **PROJECT STRUCTURE**

```
BIT-People/
    index.html
    style.css
    js/
        main.js
    README.md
```

### **GET STARTED**

To start application open index.html file.
